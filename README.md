# 🪪 Professional ID Card Generator – SAYANOX STUDIO

**Create stunning digital ID cards with AI-powered avatar cropping, themes, dual layouts, and QR code integration.**  
Perfect for LinkedIn networking & personal branding.

> 🔥 **Every card includes the watermark:** *SAYANOX • made by SAYAN* – your brand spreads with every share.

## ✨ Features

- 🖼️ **Smart Avatar Cropper** – Upload any photo, then **drag & zoom** to adjust inside a circular frame (like AI cropping).
- 🎨 **3 Premium Themes** – Gold, Neon Blue, Emerald – change card colors instantly.
- 📐 **Dual Layouts** – Switch between **Vertical** (classic) and **Horizontal** (widescreen) card designs.
- 🔗 **LinkedIn QR Code** – Auto-generates a QR that links directly to your LinkedIn profile.
- 💾 **HD PNG Export** – Download high-resolution card (3.5x scale) for social sharing.
- 📢 **One-Click LinkedIn Share** – Pre-filled post with your custom card link + hashtags.
- 📱 Fully responsive – works perfectly on mobile and desktop.

### Step-by-step:
1. **Upload your profile picture** – drag & zoom to fit perfectly.

## 🛠️ Tech Stack

- HTML5 / CSS3 (Glassmorphism + Flex/Grid)
- JavaScript (ES6)
- [html2canvas](https://html2canvas.hertzen.com/) – for high-quality image export
- [qrcode-generator](https://github.com/kazuhikoarase/qrcode-generator) – no external API required

---

## 🌐 Deployment on Vercel

1. Push this repo to GitHub.
2. Import project to [Vercel](https://vercel.com).
3. Set **Build Command** to `none` (since it's static).
4. Output directory: `.` (root).
5. Done – your live URL will be: `your-app.vercel.app`
